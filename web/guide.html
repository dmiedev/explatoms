User Guide

Here you can learn how to run and play the game.

Game Rules

The game are played by two players: red and blue.

The game is taking place on a field of squares, on which the players place colored atoms. When a square contains as many atoms as there are neighbouring squares, the atoms explode: they are spread across the neighbouring squares, simultaneously changing the color of atoms located on them.

Each player's goal is to leave no further moves to one's opponent.

More Details

There are three types of squares:

Type Properties
Location Becomes full at
Corner In the corners of the field 2 atoms
Wall Beside the walls (excluding the corners) 3 atoms
Normal The rest of the squares 4 atoms

The way that one can leave no moves to one's opponent is simple enough: once there are a lot of atoms on the field, the probability of "explosion loops" (i.e. atoms exploding infinitely) fairly increases.

Running the Program

Unfortunately, the only way to run the game for now is by building from source.

  1. Install PyGObject.
    • Follow the steps from here to install PyGObject.
    • You don't need to create any scripts as it says in the instruction, follow only the steps related to the installation.
  2. Windows users: copy the source code directory to C:\msys64\home\your-username.
  3. Navigate to the source code directory (e.g. cd explatoms).
  4. Execute python3 explatoms.

Using the Program

The user interface of the game is quite intuitive.

Once you run the game, you will see a window with the field of squares. Field's default size is medium (8x8).

You can start playing instantly. The blue player always goes first. Click on a square where you want to place your atom and now let your opponent make a move.

You can also create a smaller or a bigger field by clicking the menu button at the top left and selecting "New Game". In the new game dialog you can select the desired field size.

When the game finishes, you will see a dialog that says who won.

To start a new game, you have to create a new field by clicking the menu button at the top left and selecting "New Game".
